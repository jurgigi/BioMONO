# BioMONO_en: A corpus of bioinformatics research articles

This repository provides the codebase used to (i) build a corpus of bioinformatics research articles and (ii) extract multiword expressions (MWEs) from that corpus. The corpus itself is distributed separately (see **Corpus access** below).

## Corpus access

The corpus is available here: **[LINK TO CORPUS]**  
https://doi.org/10.6084/m9.figshare.31215955
